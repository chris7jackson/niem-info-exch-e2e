NEICE ICPC Process

This is a catalog file for the NEICE ICPC IEPD. State welfare agencies exchange these messages when dealing with the placement of children across states. This IEPD describes 6 individual messages which are described in the master documentation. These exchanges are actually based on PDF forms which the states exchange with each other. These artifacts are also included in this package.
